# Pong
A remake of the classic. Written for the web, mostly in Elm.

## Running
Opening `dist/index.html` in a web browser should suffice. Otherwise you can run `./serve.sh` to start a simple web server.

## Dependencies
### Game
- A web browser

### Server script
- Python 3

### Building
- Elm 0.19
- GNU Make

### Testing
- Elm-test 0.19.1-revision12
